Ingeniero de Software Senior

Tech Solutions Inc. · Remoto

Descripción del puesto

Tech Solutions Inc. está buscando un Ingeniero de Software Senior altamente motivado y experimentado para unirse a nuestro equipo remoto. El candidato ideal tendrá una sólida formación en desarrollo de software, con experiencia en el diseño, desarrollo e implementación de soluciones escalables y de alto rendimiento. Como ingeniero de Software Senior, usted será responsable de liderar proyectos de desarrollo, mentorizar a ingenieros junio y colaborar con equipos multifuncionales para entregar productos de software de alta calidad.

Responsabilidades

  • Desarrollar y mantener aplicaciones web utilizando tecnologías modernas.
  • Colaborar con equipos multifuncionales para definir, diseñar y enviar nuevas características.
  • Escribir código limpio, escalable y eficiente.

Requisitos

  • Licenciatura en Informática o campo relacionado.
  • Mínimo 5 años de experiencia en desarrollo de software.
  • Experiencia con frameworks de JavaScript (por ejemplo, React, Angular, Vue.js).
  • Familiaridad con metodologías ágiles y herramientas de control de versiones (por ejemplo, Git).

Acerca de la empresa

Tech Solutions Inc. es una empresa innovadora que se centra en la creación de soluciones tecnológicas de software de vanguardia para diversas industrias. Estamos comprometidos con el fomento de un entorno de trabajo colaborativo e inclusivo donde cada empleado pueda prosperar y crecer profesionalmente.
Ofrecemos salarios competitivos, beneficios integrales y oportunidades de desarrollo profesional continuo.
